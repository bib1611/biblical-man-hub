# taches-cc-resources

A collection of Claude Code resources, prompts, and templates for task management.

## Installation

```bash
# Add the marketplace source
npx tsx scripts/plugin-cli.ts marketplace add glittercowboy/taches-cc-resources

# Install the plugin
npx tsx scripts/plugin-cli.ts plugin install taches-cc-resources
```

## Contents

### Prompts
- **tasks.md** - Prompts for task management workflows including task breakdown, priority assessment, progress tracking, and completion criteria.

### Templates
- **project.md** - Project planning and organization templates with sections for goals, scope, tasks, dependencies, and risks.

## Usage

Access these resources through the Biblical Man Hub plugin system or reference them directly in your workflows.
